# 🛰️ Nexus Sentinel

Multi-cloud monitoring dashboard for the Nómada Inmortal architecture.

- Monitors: GitHub Codespaces, Gitpod, Google Cloud Shell, HuggingFace, Oracle Cloud, Supabase
- Runs health checks every 5 minutes
- Keep-alive endpoint prevents Space from sleeping
- Dashboard shows real-time status of all providers
